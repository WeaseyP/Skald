## PannerNode
### Purpose
Represents a panner effect, which positions an audio signal in the stereo field.

### Props / Inputs
| Prop      | Type           | Description                                  |
|-----------|----------------|----------------------------------------------|
| `data`    | `object`       | An object containing node data, including an optional `label`. |

### Emitted Events / Outputs
- None

### Dependencies
- `React`
- `reactflow`
