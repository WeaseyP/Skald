## InstrumentNode
### Purpose
Represents an instrument as a node within a React Flow graph.

### Props / Inputs
| Prop      | Type           | Description                                  |
|-----------|----------------|----------------------------------------------|
| `data`    | `object`       | An object containing data for the node, including a `name` property. |

### Emitted Events / Outputs
- None

### Dependencies
- `React`
- `reactflow`
