# Skald v0.1.0

Skald v0.1.0 is the first public game-production preview. The core visual audio workflow is implemented, and this release is intended for making music and sound effects while bugs and rough edges are identified in real use.

## Included

- Visual instrument and effect graph editing
- Sequencing, multi-track export, parameter locks, and project save/load
- Live audio preview and hot-swapping
- Dependency-free Odin source generation
- Organized import examples for sound effects, instruments, and songs
- Windows installer and portable ZIP
- Electron 43 runtime with zero production dependency audit findings

## Known limitations

- This is a prerelease: project formats and generated APIs may change before 1.0.
- Windows x64 is the supported packaged platform.
- The application is unsigned; Windows SmartScreen may display a warning.
- The package currently uses Electron's default icon.
- There are likely bugs. Please report reproducible issues with the saved project and exact steps when possible.

## Install

Download the Setup executable for a normal per-user install, or the win32-x64 ZIP for a portable copy. SHA256SUMS.txt can be used to verify the downloads.
